CSV Format Requirements

Required columns:

  • Tweet ID - Unique identifier for each tweet
  • Likes - Number of likes received
  • Retweets - Number of retweets received
  • Replies - Number of replies received

Optional columns:

  • Tweet text - The actual tweet content
  • Posted date - When the tweet was posted
  • Mentions - Number of mentions received

Supported date formats:

  • 2024-01-15 14:30:00
  • 2024-01-15
  • 01/15/2024
  • 15/01/2024

How to Export from X Analytics

  1. Go to X Analytics
  2. Navigate to "Tweets" section
  3. Select the date range you want to analyze
  4. Click "Export" and select "CSV" format
  5. Download the file and upload it here
